# AzPolicyLens.Discovery Module

## Description

Generate Azure Policy documentation in Markdown format. This module is offers the discovery component for the AzPolicyLens module.

## AzPolicyLens.Discovery

### [Invoke-AzplEnvironmentDiscovery](Invoke-AzplEnvironmentDiscovery.md)

Perform Azure environment discovery and collect required data for the Policy Documentation.

### [New-AzplEncryptionKey](New-AzplEncryptionKey.md)

Create AES encryption key and Initialization Vector for encrypting the environment discovery file for the Azure Policy Documentation.
